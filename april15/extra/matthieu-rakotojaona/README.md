# gochallenge2 

This is a working solution for the 2nd gochallenge

To run, do this:

```
$ go build

# in 1st terminal
$ ./gochallenge2 -l 8080

# in 2nd terminal
$ ./gochallenge2 8080
# this terminal is now connected to 1st one, you can send messages
```
